# 课堂管理系统 — 功能介绍

> 版本：v3.2.0 | 平台：Windows 桌面端 | 技术栈：Electron + React

---

## 一、产品定位

**一站式班级数字化管理工具**，覆盖教学日常的积分、考勤、作业、值日、通知、课后管理等全流程，支持教室大屏展示和学生浏览器端互动。

---

## 二、核心功能模块

### 1. 班级看板

- 教室大屏一目了然：当天值日班委、考勤统计、小组排名、作业完成情况、扣分记录等
- 模块可拖拽排序，自定义展示内容
- 附带**桌面便签**（独立小窗口，始终置顶显示关键信息）

### 2. 小组积分

- 按小组管理积分，支持加分/扣分操作
- 实时排名展示，适合课堂投屏竞争
- 支持小组成员管理、分组调整

### 3. 小组植树（游戏化激励）

- 每个小组拥有一棵专属植物（樱花、向日葵、竹子、梅花、薰衣草、银杏、橘子树、玫瑰，共8种）
- 积分转化为浇水/施肥/除虫等养护行为
- 植物从种子 → 幼苗 → 小树 → 大树 → 开花，共5个成长阶段
- SVG 动态绘制，视觉效果精致

### 4. 学生管理

- 学生信息录入与编辑
- 按小组分配管理
- 支持批量导入

### 5. 个人积分

- 每位学生独立积分，记录加分/扣分流水
- 积分流水带原因、时间戳
- 支持手动调整和排名展示
- 累计学习积分用于排名决胜

### 6. 成长记录

- 长期记录学生的成长轨迹
- 支持按时间段查询（三年内任意区间）
- 密码保护的清空操作

### 7. 值日管理

- 管理每日值日学生分工
- 值日完成状态跟踪（已完成/未完成/免除）
- 学生标注来历（如"借调"标记）
- 密码保护的重置按钮

### 8. 班级轮值

- 按周轮换的班级岗位排班表
- 自动生成轮值日历
- 支持自定义岗位

### 9. 作业管理

- 按日期记录学生作业完成情况
- 标记已交/未交/免交状态

### 10. 每日考勤

- 按日期窗口进行考勤
- 记录出勤、迟到、缺勤、请假等状态
- 统计数据汇总到看板

### 11. 午餐午休

- 午餐午休纪律管理
- 标记表现良好/违纪学生

### 12. 每日一练

- 学生每日练习签到管理
- 按名册追踪完成情况

### 13. 宝龙币（虚拟货币激励）

- 小组为单位的虚拟货币系统
- 支持发放、消费、查询余额

### 14. 数学作业等级

- 专项记录数学作业评级（A/B/C/D）
- 按学生统计等级分布

### 15. 班级通知

- 教师发送通知 → 教室大屏弹窗展示
- 支持全屏/顶部横幅两种展示模式
- 支持图片附件
- 通知紧急度设置（普通/重要/紧急）
- **确认模式**：指定学生需要点击确认已读
- 通知历史记录和已读状态追踪

### 16. 留言板

- 教师通过浏览器端输入学生留言（学生姓名+内容）
- 教室大屏端展示为**便利贴墙**效果
- 支持多标签分类（建议/感谢/心愿/其他）
- 支持图片上传（多图+粘贴）
- 支持自定义字体颜色和大小
- 支持设置过期时间，自动清理
- 防止学生在教室端随意编辑

### 17. 课后管理（三合一）

#### 小组团建
- 按小组发起团建活动
- 倒计时机制 + 学生浏览器签到
- 多组可同时进行，独立状态

#### 罚抄管理
- 记录待罚抄学生
- 学生通过浏览器端签到确认"已抄完"
- 支持手动催促通知提醒（自动附加"来办公室"）

#### 延时续费
- 管理放学后延时服务学生
- 签到/签退状态追踪
- 密码保护的重置操作

## 三、系统特性

### 远程访问

| 方式 | 说明 |
|------|------|
| **局域网（LAN）** | 同一网络下的设备通过浏览器访问，学生可用手机/平板签到 |
| **公网隧道** | 通过 Cloudflare Tunnel 映射到固定域名，支持从办公室远程管理教室系统 |

### 桌面便签看板

- 独立的迷你窗口，始终置顶
- 展示当日关键数据（积分排名、值日安排等）
- 不影响主窗口操作

### 安全机制

- 统一密码保护敏感操作（修改值日、清空记录、强制结束等）
- 教室端（Electron）只展示，管理操作通过浏览器端完成
- 防篡改：教室大屏无法直接编辑留言板等内容

### 自动更新

- 基于 Gitee Release 的 OTA 自动更新
- 检测到新版本自动提示安装

### 数据管理

- 本地 SQLite 数据库，数据完全自主
- 支持数据导出（学生信息 + 历史记录分开导出）
- 支持数据导入恢复
- 旧版本数据自动检测与迁移

## 四、使用场景

| 场景 | 功能组合 |
|------|----------|
| 课堂教学 | 小组积分 + 班级看板投屏 + 植树游戏化 |
| 作业管理 | 作业管理 + 数学等级 + 每日一练 |
| 班级日常 | 考勤 + 值日 + 轮值 + 午餐午休 |
| 师生互动 | 留言板 + 通知 + 团建签到 |
| 课后管理 | 罚抄 + 延时续费 + 团建 |
| 远程办公 | 公网隧道 + 浏览器端管理 |

## 五、技术亮点

- **Electron 桌面应用**：原生窗口体验，开机即用
- **本地优先**：数据存储在本地，无需联网也能使用
- **React + TypeScript**：现代前端架构，维护性强
- **Cloudflare Tunnel**：零配置公网穿透，固定域名访问
- **SVG 动态渲染**：植树系统8种植物×5阶段，纯代码绘制
- **framer-motion 动画**：流畅的拖拽排序和过渡效果
