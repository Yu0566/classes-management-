# 课堂管理系统 — 功能介绍（汇报版）

---

## 产品定位

一站式班级数字化管理桌面应用，覆盖教学日常全流程，支持教室大屏展示与师生远程互动。

---

## 功能概览

### 积分激励体系

- **小组积分**：课堂实时评分，投屏竞争激励
- **个人积分**：每位学生独立积分档案，记录完整流水
- **小组植树**：积分转化为养护行为，植物从种子到开花五阶段成长（游戏化）
- **宝龙币**：小组虚拟货币奖励机制

### 日常管理

- **班级看板**：教室投屏一目了然，模块可拖拽自定义
- **每日考勤**：出勤/迟到/缺勤/请假，统计自动汇总
- **值日管理**：每日分工、完成状态追踪
- **班级轮值**：按周自动轮换岗位排班
- **午餐午休**：纪律管理与记录

### 作业管理

- **作业管理**：按日期记录提交情况
- **数学作业等级**：A/B/C/D 评级统计
- **每日一练**：签到式练习追踪

### 师生互动

- **班级通知**：教师发送 → 教室大屏弹窗，支持图片、紧急度、确认已读
- **留言板**：学生留言 → 教室便利贴墙展示，支持分类与图片
- **成长记录**：长期记录学生成长轨迹，支持按时段查询

### 课后管理

- **小组团建**：倒计时 + 学生浏览器签到
- **罚抄管理**：学生线上签到确认完成，支持催促通知
- **延时续费**：放学后延时服务签到管理

---

## 系统特色

| 特色 | 说明 |
|------|------|
| 教室大屏适配 | 看板投屏、通知弹窗、便利贴墙，专为教室场景设计 |
| 远程管理 | 局域网浏览器访问 + 公网固定域名，办公室也能管教室 |
| 学生端互动 | 学生通过手机/平板浏览器完成签到、确认等操作 |
| 数据安全 | 本地存储、密码保护、自动备份，数据完全自主可控 |
| 自动更新 | 新版本自动推送安装，维护零成本 |

---

## 应用场景

- **课堂教学**：积分竞争 + 看板投屏 + 植树激励，提升学生参与度
- **班级日常**：考勤、值日、轮值一键管理，减少人工记录负担
- **家校沟通**：通知确认已读 + 留言板互动，信息传达有保障
- **远程办公**：公网访问支持跨地点管理，不受物理位置限制
